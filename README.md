## Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
